# Trading App

A full-stack trading application built with React and Node.js.

## Getting Started

1.  **Clone the repository:**

    bash
    git clone <repository-url>
    

2.  **Navigate to the client directory:**

    bash
    cd trading-app/client
    

3.  **Install client dependencies:**

    bash
    npm install
    

4.  **Start the client development server:**

    bash
    npm run dev
    

5.  **Open a new terminal and navigate to the server directory:**

    bash
    cd trading-app/server
    

6.  **Install server dependencies:**

    bash
    npm install
    

7.  **Start the server:**

    bash
    npm run dev
    

8.  **Open your browser and go to `http://localhost:5173` to view the application.**
